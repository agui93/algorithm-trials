**动态规划三要素**

- 重叠⼦问题
- 最优⼦结构 (子问题独立)
- 状态转移方程
    - 确定base case
    - 确定状态: 即原问题和⼦问题中会变化的变量
    - 确定选择: 导致状态产⽣变化的选择
    - 确定定义: 备忘录 函数
